武器收藏

效率工具​

  • snipaste 截图+贴图
  • uTools 可插拔综合工具+超级面板
  • fiddler 抓包大哥大
  • charles 抓包工具
  • proxyman 抓包工具 mac(基础功能免费)
  • parsec 免费无人值守远程控制工具，很快很稳
  • ScreenToGif 免费开源的录制 GIF 工具

在线工具(打开网页即用)​

  • cubic-bezier 贝塞尔曲线运动预览
  • easings 贝塞尔曲线预制函数 + 场景预览
  • tinypng 压缩图片的一哥
  • css_sprite 根据精灵图生成 css 代码 - 脚本之家在线工具
  • sprite-generator 散图打包生成精灵图神器
  • gif-to-jpg 转换 GIF 动画文件的每一帧至 JPG 文件
  • jsont 在线格式化 json，提供多种展示方式(还是推荐用 uTools 装插件，这个临时用可以)
  • monkeytype 在线练打字、比金山打字通还好用
  • alltoall 在线文件格式转换平台
  • paletton 配色方案设计师(简单粗暴算法配色)
  • dribbble 设计灵感
  • sukoutu-ps 在线 PS 小工具(当然能用 Figma 还是不用这个，这个主要打开快) - 速抠图
  • svg.wxeditor SVG 在线编辑
  • loader-generator CSS 简易加载动画生成器
  • codepen 前端炫酷花活大型表演秀
  • uiverse 纯 CSS 炫酷组件
  • neumorphism 预览 box-shadow 生成代码
  • dotnetfiddle 在线运行 C# 代码，还提供 NuGet 集成，不想装环境跑 Demo 神器
  • split-pdf 在线拆分 PDF，提取想要的页面
  • chalk.ist 在线制作好看的源代码图片，做封面、README 介绍神器
  • rive.app 构建可在任何地方运行的交互式动画
  • free-tex-packer 免费纹理打包器，也可分割表工具
